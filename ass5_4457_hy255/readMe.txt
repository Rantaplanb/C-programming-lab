1d) H συναρτηση main περιεχει 19 εντολες assembly.
    Στην x86 assembly, οι εντολες εχουν μεταβλητο μεγεθος (απο 1 εως 8 bytes),
    αλλα ξερουμε οτι οι εντολες αποθηκευονται σε συνεχομενες θεσεις μνημης,
    συνεπως αρκει να αφαιρεσουμε την διευθυνση της 1ης εντολης της main απο την
    διευθυνση της αμεσως επομενης εντολης μετα το τελος της συναρτησης μας ωστε
    να βρουμε το πληθος των bytes μνημης που καταλαμβανουν οι εντολες μας.
    0x8048437 - 0x80483fb -> 60 bytes!

    Για να βρω το offset: 0x8048417 - 0x80483fb = 28 bytes offset
 
1e) p(int)S -> $1 = 500500
    p (int *)&S -> $2 = (int *) 0x80496a0
    p (char *)&Msg -> $3 = 0x80496a4 "The sum from 0 to %d is %d\n"
    
    Αφοτου αλλαξουμε την τιμη που περιεχεται στην διευθυνση S,
    τυπωνεται το τελικο μηνυμα: "The sum from 0 to 1000 is 99".
